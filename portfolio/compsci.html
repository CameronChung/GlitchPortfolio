Bot Quest



This game was based off the idea of "Where's Waldo?", using the clicker game framing in the course provided by Code.org, I incorporated the robot theme. With fun accessorized bots, the objective is ti find the real bot in clusters of fake ones. Find the real bots and pass all the levels before time runs out!
